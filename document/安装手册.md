#### 配置服务开机自启动

在/etc/systemd/system/clouddisk.service中写入

```
[Unit]
Description=just big homework to build a remote disk
[Service]
Type=simple
ExecStart=clouddisk
[Install]
WantedBy=multi-user.target
```

输入以下命令注册自启

```
systemctl daemon-reload
systemctl enable clouddisk
```
